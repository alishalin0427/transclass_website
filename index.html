核心特色

即時語音識別

採用 Azure Speech-to-Text API，支援中文、英文、日文、越南文等多種語言輸入，識別準確率高達90%以上。

多語言翻譯

整合 Azure Translator API，支援100+種語言互譯，特別優化學術術語翻譯，提供流暢自然的課堂內容轉譯。

課程紀錄管理

自動生成課程摘要與翻譯紀錄，支援多格式匯出功能，方便學生課後複習與教師教學評估。

系統概述

TransClass 是一套專門為國際學生設計的多語言即時課堂翻譯系統。

系統採用 Microsoft Azure 的先進 AI 技術，能夠在課堂進行中即時轉換語音並翻譯成多種語言，幫助外籍學生克服語言障礙。

系統整合了語音識別、機器翻譯、資料庫管理等多項先進技術，提供穩定可靠的雲端解決方案。透過直觀的使用者介面，讓教師和學生都能輕鬆使用。

  • 100+ 語言支援

Microsoft Azure Translator API

  • 100+ 語言互譯
  • 領域適應性優化

資料管理層
