打破語言隔閡的智慧課堂翻譯系統

TransClass 運用 Azure AI 技術，為國際學生提供即時多語言課堂翻譯，解決中文授課的理解障礙，提升學習參與度與成效。

系統特色 觀看示範

國際學生的語言困境

根據教育部統計，台灣有超過13萬名國際學生，其中65%反映中文授課理解困難，特別是在專業課程中面臨術語障礙與課堂參與度低的問題。

  • 專業術語造成理解斷層
  • 課堂討論參與度不足
  • 學習挫折感影響學業表現

核心技術特色

即時語音識別

採用 Azure Speech-to-Text API，支援中文、英文、日文、越南文等多種語言輸入，識別準確率高達90%以上。

多語言翻譯

整合 Azure Translator API，支援100+種語言互譯，特別優化學術術語翻譯，提供流暢自然的課堂內容轉譯。

課程紀錄管理

自動生成課程摘要與翻譯紀錄，支援多格式匯出功能，方便學生課後複習與教師教學評估。

系統技術架構

專題摘要

隨著台灣高等教育積極推動國際化政策，外籍學生人數逐年攀升。根據教育部統計，目前在台外籍學生已超過13萬人，來自東南亞、歐美、日韓等不同語言文化背景。然而，中文授課課程對於非中文母語學生而言，往往形成顯著的學習障礙。

本系統採用先進的語音識別與機器翻譯技術架構，整合 Microsoft Azure Speech-to-Text API 進行高精度中文語音識別，採用 Azure Translator API 實現多語言翻譯功能，支援 100+ 種語言。系統後端採用 PostgreSQL 進行資料管理，提供完整的使用者管理機制、翻譯歷史紀錄功能與個人化設定儲存。

系統為解決跨文化高等教育中的語言障礙問題提供了創新且實用的技術解決方案，具有良好的推廣應用價值與發展潛力。

系統優點與限制

系統優點

  • ✓ 多語言支援廣泛，滿足多國籍學生需求
  • ✓ 使用者介面簡潔友善，易於操作

系統限制

  • • 對網路連線品質依賴度高

  • • 目前支援中英日越文輸入
  • • 離線環境無法使用
